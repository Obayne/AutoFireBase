AutoFire Keyboard Shortcuts

Global
- New: Ctrl+N
- Open: Ctrl+O
- Save: Ctrl+S
- Save As: Ctrl+Shift+S
- Undo: Ctrl+Z
- Redo: Ctrl+Y (or Ctrl+Shift+Z)
- Delete: Del
- Select All: Ctrl+A
- Zoom In: Ctrl++
- Zoom Out: Ctrl+-
- Fit View to Content: F2

View / Toggles
- Toggle Crosshair: X

Draw Tools (single letters)
- Line: L
- Rectangle: R
- Circle: C
- Polyline: P
- Arc: A
- Wire: W
- Text: T

Modify Tools
- Trim: Tr
- Extend: Ex
- Fillet: Fi
- Move: Mo
- Copy: Co
- Rotate: Ro
- Mirror: Mi
- Scale: Sc
- Chamfer: Ch
- Offset: O

Measure / Dimension
- Measure: M
- Dimension: D

Misc
- Cancel Active Tool / Escape: Esc
